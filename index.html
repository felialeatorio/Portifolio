Projetos

Sistema de Gerenciamento de Estoque

Como meu primeiro projeto desenvolvi um sistema de gerenciamento de estoque escalável utilizado por mais de 200 lojas para monitorar e controlar seus inventários em tempo real. O sistema foi projetado para lidar com grandes volumes de dados, oferecendo uma interface de fácil navegação e relatórios precisos para gerentes e operadores.

No back-end, utilizei PHP e MySQL para a estruturação do banco de dados, com foco em consultas otimizadas que garantissem a velocidade de resposta, mesmo com altas demandas de usuários simultâneos. O front-end foi construído com JavaScript e jQuery, assegurando que as interações fossem rápidas e fluidas, com atualizações dinâmicas via AJAX.

Plataforma EAD

Desenvolvi uma plataforma de ensino a distância (EAD) completa, utilizada por mais de 1.200 alunos e educadores. A plataforma foi construída para oferecer uma experiência educacional robusta e flexível, permitindo o cadastro e gestão de cursos, materiais didáticos, e fóruns de discussão.

Atuei no desenvolvimento tanto do back-end, utilizando PHP e o framework Laravel para a criação de APIs RESTful, quanto no front-end com Tailwind CSS para garantir uma interface intuitiva e responsiva. A plataforma inclui recursos de autenticação, controle de acesso baseado em perfis, geração de relatórios dinâmicos, e integração com APIs de pagamento para matrículas.

Site de curso de Coreano

Desenvolvi o site da Coreano Ao Vivo para uma escola de idiomas, projetado para fornecer informações sobre cursos, professores e inscrições. O site inclui páginas para detalhes dos cursos, um formulário de contato, informações dos professores e uma seção de FAQs. O objetivo principal era criar uma presença online clara e acessível para os alunos interessados em aprender mais sobre as ofertas da escola.

O site foi desenvolvido utilizando PHP para a criação do back-end e JavaScript para melhorar a interatividade do front-end. A interface foi construída com BootStrap para garantir que o design fosse responsivo e atraente em dispositivos móveis e desktops. A solução final proporcionou uma navegação intuitiva e facilitou o acesso a informações para interressados no curso.

Aplicativo Mobile de Tarefas e Notificações

Estou Desenvolvendo um aplicativo mobile de gerenciamento de tarefas e notificações usando Flutterflow. O app permite que os usuários criem listas de tarefas, estabeleçam prioridades, e recebam notificações para prazos importantes.

O aplicativo está sendo projetado com foco em usabilidade, oferecendo uma experiência de usuário fluida em dispositivos Android e iOS. A integração com serviços de nuvem também permite sincronização em tempo real das tarefas entre diferentes dispositivos, o projeto esta em desenolvimento ainda.

Sobre Mim

Sou um desenvolvedor de software com mais de 3 anos de experiência no desenvolvimento de soluções web e mobile. Minha especialidade é trabalhar com tecnologias modernas para criar aplicações escaláveis e de alta performance que atendam às necessidades de negócios em diversos setores. Tenho ampla experiência no desenvolvimento full stack, utilizando PHP e Laravel no back-end para criar APIs robustas e arquiteturas MVC, além de JavaScript no front-end para construir interfaces dinâmicas e responsivas.

Além disso, sou adepto de boas práticas de desenvolvimento, como programação orientada a objetos (POO), testes automatizados e integração contínua (CI/CD). Essas práticas garantem que o código seja limpo, escalável e fácil de manter. Também tenho experiência com SEO e otimização de UX, o que me permite criar aplicações não apenas funcionais, mas também voltadas para melhorar a conversão e a experiência do usuário final.

Nos últimos anos, expandi minhas habilidades para o desenvolvimento mobile com Flutter e FlutterFlow, permitindo que eu criasse aplicativos móveis que proporcionam uma experiência de usuário fluida e consistente em diferentes dispositivos.

Minha abordagem ao desenvolvimento é sempre focada em resolver problemas complexos de maneira eficiente e criar soluções que sejam flexíveis o suficiente para acompanhar o crescimento do negócio. Trabalhando em equipe, gosto de compartilhar conhecimento e colaborar para encontrar as melhores soluções técnicas e estratégias de produto. Com uma mentalidade ágil, estou sempre em busca de aprimorar processos e adotar novas ferramentas que possam acelerar o desenvolvimento sem comprometer a qualidade.
